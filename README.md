## Installation

1. Upload the `my-rest-blog` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to "API Docs" in the WordPress admin menu to access the interactive documentation

## API Endpoints
GET /wp-json/myplugin/v1/hello?name=YourName
GET /wp-json/myplugin/v1/posts?per_page=5&category=news
